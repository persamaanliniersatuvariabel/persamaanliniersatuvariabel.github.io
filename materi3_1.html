Pertidaksamaan Linier Satu Variabel

2. PtLSV dalam Berbagai Bentuk dan Variabel

Dalam pertidaksamaan linier satu variabel dikenal beberapa istilah, antara lain ketidaksamaan, pertidaksamaan, dan ptlsv.

Ketidaksamaan

  Ketidaksamaan adalah pernyataan (kalimat tertutup) yang memuat hubungan (relasi) tidak sama dengan yang meliputi kurang dari atau lebih dari (<, ≤, >, ≥).
Contoh :

 1. -2 + 4 > 1
 2. 4 × (-2) > -6

Pertidaksamaan

 Pertidaksamaan adalah kalimat terbuka yang menggunakan hubungan relasi tidak sama dengan.

 1. 2y + 1 > 5
 2. 3z – 2 ≤ 7

PtLSV

  Pertidaksamaan linier satu variabel adalah pertidaksamaan yang hanya memuat satu variabel dengan pangkat tertinggi variabelnya satu. Bentuk pertidaksamaan linear satu variabel dalam x sebagai berikut.

ax + b < 0 atau ax + b ≤ 0 atau ax + b > 0 atau ax + b ≥ 0

Keterangan:
x = variabel
a = koefisien variabel x dengan a ≠ 0
b = konstanta

Contoh:

  1.  4x – 3 < 0
  2.  5x + 1 > 7
  3.  $\frac{1}{2}$x − 3 ≥ $\frac{2}{3}$x
  4.  $\frac{2}{5}$x - 1 ≥ $\frac{x}{2}$ + 5

Bentuk PtLSV diatas semuanya memiliki sebuah variabel berpangkat satu, yaitu x. PtLSV juga dapat dituliskan dalam berbagai variabel, diantaranya sebagai berikut.

  1.  2 (y + 3) + 5y ≥ 7 – (y -1)
  2.  $\frac{p - 1}{3}$ + 1 < $\frac{p + 2}{5}$
  3.  n – 3(n + 1) > n
  4.  $\frac{m - 1}{m + 1}$ ≤ 0
